# Проект для сортировки файлов

**Цель проекта:** Отработка навыков работы с файловой системой, GUI, создание удобного инструмента для повседневного личного использования

---

## Основной функционал

- **Сортировка файлов**:
	- По расширениям (`.pdf`, `.jpg`, `.docx` и т.д.)
	- По ключевым словам в названии (например: `"отчет"`, `"важно"`)
- **Конфигурация перемещений**:
	Процесс происходит через работу с таблицей, в которой можно добавлять и удалять строки. Файлы, конфигурации представляют собой простые текстовые файлы. 
- **Массовое переименование файлов:**
	Добавление ключа к названиям выбранных пользователем файлов в фиксированном формате. 
- **Синхронизация:**
	Простой алгоритм синхронизации файлов, который сравнивает два списка файлов в двух разных директориях.
